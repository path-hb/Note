# 파이썬 메모장 설계

## 기능 정의
### 1. 문서의 현재 상태 확인:
	텍스트 입력 내용을 처음부터 끝까지 검사하여
	마지막으로 저장된 상태에서 변경이 있으면 
            저장여부를 물어봄(메세지박스)

### 2. 새 파일:
	문서 상태 확인
	현재 화면의 전체 내용을 지움
	프로그램 타이틀을 "이름없는 메모장"으로 바꿈

### 3. 불러오기:
	문서 상태 확인
	filedialog 모듈로 파일 경로 확인
	현재 화면의 전체 내용을 지움
	경로의 파일을 화면에 삽입
	프로그램 타이틀을 "{불러온 파일명} - 메모장"으로 바꿈

### 4. 저장하기:
	filedialog 모듈로 저장할 경로와 파일명 설정. 기본형은 *.txt
	현재 화면의 전체 내용을 경로의 파일에 씀
	프로그램 타이틀을 "{저장한 파일명} - 메모장"으로 바꿈

### 5. 스크롤:
	화면 우측에 세로 스크롤 배치
	스크롤이 움직이면 화면도 같이 움직이고,
	화면을 움직여도 스크롤이 같이 움직임
